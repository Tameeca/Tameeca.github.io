Geometric Animation

My first programming project, concluding the 2 week Programming Bootcamp at City (2020). This Processing project demonstrates a comprehensive understanding of the use of the for loop, shapes, and interaction with the sketch. This sketch displays 5 different vertexes moving at different points, forming a shape. It also allows you to interact with the sketch by pressing the keys "w,s,a,d,c" to move the shape up, down, left, right, and centre, respectively. You can also move the shape with your mouse.

Check out the code on Github: https://github.com/Tameeca/adbt149.git
